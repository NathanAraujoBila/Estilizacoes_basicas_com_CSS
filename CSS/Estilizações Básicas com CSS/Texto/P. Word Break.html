Propriedade word-wrap

Define, se caso uma palavra for muito grande pr caber um uma linha, se ela esta quebrada em qualquer caractere para que uma parte dela fique na linha e o restante na proxima, ou se nao, toda palavra deve ficar na mesma linha e ultrapassar os limtes que decidimos para o elemento.
